hexo+github搭建个人博客

2024-07-12
hexo攻略
1.1k words

说明

使用win+X，点击Windows PowerShell（管理员）进入PowerShell
(vim可选)不用vim就直接打开文件即可
能查看版本不报错就是说明安装好了

下载nodejs

进入官网下载nodejs，在PowerShell中使用以下命令安装hexo

搭建默认网页

打开浏览器打开http://localhost:4000/
至此可以看到已经搭建好了默认的网页

编辑blog

按i进入编辑模式，随便输入一点内容
编辑完成后按esc
输入:wq保存退出
至此你的post搭建完成

至此可以看到第一篇博客

上传github pages

修改最后为:
type: git
repo: https://github.com/oldsixyaoyao/oldsixyaoyao.github.io.git
branch: master
至此配置完成

在github中建立一个仓库，名称为oldsixyaoyao.github.io

注：第一次用git可能要你登录账号密码
至此可以用oldsixyaoyao.github.io访问你的网站了

切换主题yilia

把theme: landscape改成theme: yilia
再重新上传到github仓库就可以了
